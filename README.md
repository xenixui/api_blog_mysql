# API Blog - MySQL

API REST para una aplicación de blog desarrollada con Node.js y MySQL.

## Stack

* Node.js
* Express
* MySQL

## Instalación

```bash
npm install
```

## Variables de entorno

Crear un archivo `.env` en la raíz del proyecto:

```env
PORT=3000
DB_HOST=localhost
DB_USER=root
DB_PASSWORD=
DB_NAME=blog_db
```

## Ejecutar proyecto

```bash
npm run dev
```

## Endpoints

Las pruebas de endpoints están en:

```bash
peticiones.rest
```
